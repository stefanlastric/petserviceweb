05.02.2019




Description

	The purpose of Pet Services application is to provide a centralized place for acquiring various services related to pets such as adoption, medical services, pet sitting, grooming and more. The user will be able to choose from a pool of given veterinaries, grooming parlors and adoption service that are enlisted within the app. Users will be able to view a list of adoptable pets along with pictures, videos and description of eligible animals; they would be to make an appointment with adoption agencies. Users can take an interactive test to determine which pet would suit them the best. Also the application will also have a section where lost pets will be posted in order to help pet owners to find pets faster. The application will be for Android based devices.
